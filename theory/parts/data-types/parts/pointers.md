#### Pointers
В Go используются указатели. Вместо того, чтобы хранить значение, они хранят адрес в памяти на него.
```Go
*T - это указатель на значение типа T, его нулевое значение равно nil
```

Оператор **&** генерирует указатель на свой операнд.

```Go
i := 42
p = &i
```

Оператор Оператор **\*** обозначает значение, на которое указывает указатель.

```Go
fmt.Println(*p) // Прочитать значение, на которое указывает указатель
*p = 21 // Присвоить новое значение через указатель
```

Правильное название этого действия - разыменование или косвенное указание. В отличии от **C**, в Go нету арифметики указателей.
